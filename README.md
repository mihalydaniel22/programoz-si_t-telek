#Programozási tételek


##Elõkészítés
 - Tíz elemû egész számot tartalmazó tömb létrehozása
 - A tömb elemei:
	- 12, 7, 2, 4, 45, 99, 1, 8, 95, 50
 - A tömb az összes metódusban látható legyen
	- TömbKiírás() metódus létrehozása
	- nem ad vissza értéket
	- nincs paramétere
	- a tömb elemeit vesszõvel elválasztva egy sorban a consol-ra írja

##Összegzés
  -Osszegzés() metódus létrehozása
	 -összegzés tétele a tömbön
	 -eredmény kiírása a console-ra


##Megszámolás
	- Megszamolas() metódus létrehozása
	- Hány 3-mal soztható szám van
	- Eredmény kiírása a console-ra

##Eldöntés
	- Eldontes() metódus létrehozása
	- Van-e a tömbben 69
	- Eredmény kiírása a console-ra

##Kiválasztás
	- Kiválasztás metódus létrehozása
	- Hányadik az 50-es
	- Kiírni a helyét a tömbben

##Keresés(lineáris)
	- Kereses() metódus létrehozása
	- Van-e benne 50 és ha van akkor hányadik?
